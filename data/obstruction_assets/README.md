# Obstruction Assets

このディレクトリに障害物として表示する画像や動画を配置してください。

## 対応フォーマット

### 画像
- PNG
- JPG/JPEG
- GIF

### 動画
- MP4
- WebM

## 推奨仕様

- **サイズ**: 200x200px 〜 500x500px
- **ファイルサイズ**: 5MB以下
- **動画の長さ**: 5秒以下（ループ再生されます）

## 使用例

障害物として面白い画像や動画を追加することで、配信をより楽しくできます：

- アニメキャラクター
- ミーム画像
- 動くGIFアニメーション
- 短い動画クリップ

## 注意事項

- 著作権に注意してください
- 視聴者が不快に思う画像は避けてください
- ファイルサイズが大きすぎるとパフォーマンスに影響します
